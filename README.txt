Посмотреть на работу сервиса можно здесь cifrium-hack.ru
В случае нерабочего состояния сайта, можно запустить локально на своем компьютере для этого необходимо выполнить следующие шаги
1) git clone https://github.com/Sheesh1006/service-compose
2) docker-compose up --build -d
3) В поисковой строке любого браузера написать localhost:8000/
